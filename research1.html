What is DNS?

DNS, or Domain Name System, is the Internet's naming system that translates human-friendly domain names into IP addresses that machines use to locate each other.

History of DNS

Originally, ARPANET used a single HOSTS.TXT file to resolve names. As the network grew, Paul Mockapetris introduced DNS in 1983 to enable scalable, distributed resolution of domain names.

How DNS Works

When a user types a domain name, a recursive resolver checks its cache or queries a root server, which points to a TLD server, then to the authoritative server that returns the IP address.
