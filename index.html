Executive Summary

The Domain Name System (DNS) is a core component of the internet, enabling human-friendly names to be translated into machine-readable IP addresses.

Key Terms

  • DNS: Domain Name System
  • Resolver: Translates domain names
  • TLD: Top-Level Domain (.com, .org)
  • IP Address: Numeric address of a website
